<strong>[REDACTED - PII]</strong> вҖ” Machine-readable, type-labelled format. Best for legal documents where the type of redacted data must be documented.<br/>
<strong>[CONFIDENTIAL]</strong> вҖ” Formal and generic. Best for internal memos and reports where you don't want to label the type.

<details>
<summary><strong>How does Click-to-Reveal work?</strong></summary>
<br/>
Each redacted block in the output pane is a clickable span element. Clicking it reveals the original underlying text (highlighted in green). Clicking again re-masks it. This is essential for handling false positives вҖ” for example, if a phone number pattern matches a non-phone number sequence in your document. Each block is independently toggleable.

</details>

Crucially, it contains <strong>no actual PII data</strong> вҖ” only the summary counts. This makes it safe to store, share with compliance officers, or attach to GDPR/HIPAA documentation.
